This tutorial scene demonstrated you 3 types of screen distortion:

1, burst_wave:
key point: the distort intensity is controlled by Color Config.

2, heat_wave
key point: the distortion is changed dynamically by scrolling the Displacement Map.

3, controlled_distort
key point: the distort strength is controlled by the Advance Shader Control of EffectLayer. 


NOTE: This shader requires unity pro and it's not mobile friendly.
